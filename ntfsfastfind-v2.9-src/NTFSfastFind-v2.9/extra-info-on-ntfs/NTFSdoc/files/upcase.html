File - $UpCase (10)

Previous Next

Overview

This is a 128KB file full of capital letters. For each character in the Unicode alphabet, there is an entry in this file. It is used to compare and sort filenames.

Attributes

Type Description Name
0x10 $STANDARD_INFORMATION  
0x30 $FILE_NAME $UpCase
0x80 $DATA [Unnamed]

Layout of the File

Unnamed Data Stream

Offset Character
~ ~

~ ~

Notes

Other Information

This file allows the system to compare filenames independently of their code page.
